

>[!prp] 命题
适合含多项式形如 
>$$
>\displaystyle\int P(x)f(x)\ \mathrm{d}x
>$$
>的积分的一种分部积分法的计算简便手法。


# 1. 列表格

| 次数       | 求导栏          | 积分栏                                   |
| -------- | ------------ | ------------------------------------- |
| $0$      | $P(x)$       | $f(x)$                                |
| $1$      | $P^{(1)}(x)$ | $\displaystyle\int f(x)\ \mathrm{d}x$ |
| $\vdots$ | $\vdots$     | $\vdots$                              |
| $n-1$    | 非零导数         | $n-1$ 重积分                             |
| $n$      | $0$          | $n$ 重积分                               |

# 2. 错一位相乘再相加



| 项        | 符号                                                                          | 正负           |
| -------- | --------------------------------------------------------------------------- | ------------ |
| 第一项      | $P(x)\displaystyle\int f(x)\ \mathrm{d}x$                                   | $+$          |
| 第二项      | $P^{(1)}(x)\displaystyle\iint f(x)\ \mathrm{d}x$                            | $-$          |
| $\vdots$ | $\vdots$                                                                    | $\vdots$     |
| 最后一项     | $P^{(n-1)}(x)\displaystyle\iiint\cdots \displaystyle\int f(x)\ \mathrm{d}x$ | $(-1)^{n+1}$ |
