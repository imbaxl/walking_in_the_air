# 1. 引言

因为 [[常用导数]] 不太全面（比如 $\displaystyle\int  \dfrac{1}{\sqrt{ 1+x^2 }}\mathrm{\ \mathrm{d}}x$），所以另外做一份积分表.

# 2. 积分表

$$
\begin{aligned}
&\int 0 \, \ \mathrm{d}x = C &
&\int 1 \, \ \mathrm{d}x = x + C \\
&\int x^n \, \ \mathrm{d}x = \frac{x^{n+1}}{n+1} + C \quad\ (n \ne -1) &
&\int \frac{1}{x} \, \ \mathrm{d}x = \ln |x| + C \\
&\int e^x \, \ \mathrm{d}x = e^x + C &
&\int a^x \, \ \mathrm{d}x = \frac{a^x}{\ln a} + C \quad\  (a > 0, a \ne 1) \\
&\int \sin x \, \ \mathrm{d}x = -\cos x + C &
&\int \cos x \, \ \mathrm{d}x = \sin x + C \\
&\int \tan x \, \ \mathrm{d}x = -\ln |\cos x| + C &
&\int \cot x \, \ \mathrm{d}x = \ln |\sin x| + C \\
&\int \sec x \, \ \mathrm{d}x = \ln |\sec x + \tan x| + C &
&\int \csc x \, \ \mathrm{d}x = \ln |\csc x - \cot x| + C \\
&\int \sec^2 x \, \ \mathrm{d}x = \tan x + C &
&\int \csc^2 x \, \ \mathrm{d}x = -\cot x + C \\
&\int \sec x \tan x \, \ \mathrm{d}x = \sec x + C &
&\int \csc x \cot x \, \ \mathrm{d}x = -\csc x + C \\
&\int \frac{1}{\sqrt{1 - x^2}} \, \ \mathrm{d}x = \arcsin x + C &
&\int \frac{1}{1 + x^2} \, \ \mathrm{d}x = \arctan x + C \\
&\int \frac{1}{x \sqrt{x^2 - 1}} \, \ \mathrm{d}x = \operatorname{arcsec} |x| + C \quad (|x| > 1) &
&\int \sinh x \, \ \mathrm{d}x = \cosh x + C \\
&\int \cosh x \, \ \mathrm{d}x = \sinh x + C &
\end{aligned}
$$
$$
\begin{aligned}

&\int \frac{1}{\sqrt{x^2 + a^2}} \, \ \mathrm{d}x = \ln \left| x + \sqrt{x^2 + a^2} \right| + C \quad (a \ne 0) \\
&\int \frac{1}{\sqrt{x^2 - a^2}} \, \ \mathrm{d}x = \ln \left| x + \sqrt{x^2 - a^2} \right| + C \quad (|x| > |a|) \\
&\int \frac{1}{a^2 + x^2} \, \ \mathrm{d}x = \frac{1}{a} \arctan\left(\frac{x}{a}\right) + C \quad (a \ne 0)
\end{aligned}
$$


> **注**：所有公式中的 $C$ 表示任意常数（积分常数）。
