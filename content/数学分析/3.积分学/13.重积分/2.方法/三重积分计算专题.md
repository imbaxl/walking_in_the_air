# 1. 体积微元的几何推理



![[数学分析/3.积分学/13.重积分/2.方法/_resources/三重积分计算专题/233aba7f2385e293c58e9204b7d1152a_MD5.jpg]]
<center>二维情形的面积微元</center>


$r \ \mathrm{d}\theta$ 就是弧长。三维情形的也是同样的构建方式，通过变换到球坐标系后，对比地球的经纬度系统，固定$\ \mathrm{d}r$ 是体积微元的径向边长（外方向，直线），接下来确定极角方向边长（与经度平行，弧线）和方位角方向边长（与纬度平行，弧线）。

- **极角方向边长**就是极角的一点增量乘上半径$$r \ \mathrm{d} \varphi$$
- **方向角方向边长**就是方向角的一点增量乘上平行圈（与赤道平行）的半径
$$r \sin \varphi \ \mathrm{d}\theta$$

这两种原理的区别在于，极角方向边长就是竖下来的弧长，它只依赖 $r$，而方向角方向边长同时依赖 $r,\varphi$。如果 $\varphi=0$，则 $\theta$ 即使转一圈也还在原地.


从 [[重积分的变量代换]] 可知，变量代换能支持 $n$ 重积分的球坐标变换，虽然能通过 Jacobi 行列式知道 $$\ \mathrm{d}V=r^2 \sin \varphi \ \mathrm{d} \theta \ \mathrm{d} \varphi $$但是从几何角度推理过一次，就更不容易忘记。

# 2. 数道例题


## 2.1. 计算 $\displaystyle\iiint_{\Omega} z^{2} \, \ \mathrm{d}x \, \ \mathrm{d}y \, \ \mathrm{d}z$，其中 $\Omega$ 是由椭球面 $\dfrac{x^{2}}{a^{2}} + \dfrac{y^{2}}{b^{2}} + \dfrac{z^{2}}{c^{2}} = 1$ 所围成的空间闭区域.
?
1. 单变量 -> 考虑截面法
	1. 先积面积，外层变量视作常数：$$\dfrac{x^{2}}{a^{2}} + \dfrac{y^{2}}{b^{2}}  = 1 -  \dfrac{z^{2}}{c^{2}} $$ 此处就能得到椭圆面积，再往外层积分即可
2. 投影法的难处：对内层变量 $z$ 积分的上下限比较复杂



## 2.2. 计算 $\displaystyle\iiint_{\Omega} x \, \ \mathrm{d}V$，其中 $\Omega$ 为三个坐标面及平面 $x + 2y + z = 1$ 所围成的闭区域.
?
1. 投影法和截面法理解的例题
2. 投影法：先计算以其他两个轴为常量的定积分，再计算固定的投影面积，用在这道题会比较简单
3. 截面法：先计算以固定某个轴（比如 $x$ 轴）为常数的面积，再计算固定区间的定积分，用在这道题也是没问题，但会有点绕
	1. 假设确定 $z$ 是固定的轴，此时 $z:0 \to 1$，以 $xy$ 平面计算二重积分，先算 $y:0\to \dfrac{1-z-x}{2}$ 的积分，再算 $x: 0 \to 1-z$ 的积分，展开的时候会非常麻烦
	2. 先算 $x:0\to 1-z-2y$，再算 $y:0\to \dfrac{1-z}{2}$
	3. 不论哪个，计算起来都十分繁琐



## 2.3. $\displaystyle\iiint_{\Omega} z \, \ \mathrm{d}x \, \ \mathrm{d}y \, \ \mathrm{d}z$，其中 $\Omega$ 是由曲面 $z = x^{2} + y^{2}$ 与平面 $z = 4$ 所围成的闭区域.
?
1. 投影法：投的固定面积是 $x^2+y^2=4$ 的面积，内层函数是 $z:x^2+y^2 \to 4$，使用柱面坐标替换
	1. $z:r^2\to 4$
	2. $r:0\to 2$
	3. $\theta:0\to 2\pi$
2. 截面法：先二后一就是先算面积，算面积时z就是一个常数，因为此时是dxdy为面积微元，此时得到的面积就是 $\pi z$，直接给外层函数计算定积分即可，也是**体现截面法在单变量情形下的优势**，使用柱面坐标替换
	1. $z:0\to 4$
	2. $r:0\to \sqrt{z}$
	3. $\theta:0\to2\pi$



## 2.4. 计算 $\displaystyle\iiint\limits_{\Omega} z^2 \, \ \mathrm{d}x\ \mathrm{d}y\ \mathrm{d}z$ ， 其中 $\Omega = \left\{ (x, y, z) \mid x^2 + y^2 + z^2 \leq 1 \right\}$ .
?
1. 考虑用球面坐标替换
	1. $\theta:0\to 2\pi$
	2. $r:0\to 1$
	3. $\varphi:0\to \pi$
	4. $\displaystyle\iiint\limits_{\Omega} z^2 \, \ \mathrm{d}x\ \mathrm{d}y\ \mathrm{d}z=\displaystyle\int_{0}^{2\pi}\ \mathrm{d}\theta \displaystyle\int_{0}^\pi \ \mathrm{d} \varphi \displaystyle\int_{0}^1 r^2 \cos^2 \varphi \ r^2\sin \varphi  \ \mathrm{d}r$
	5. 若使用轮换对称性，原积分化为 $$\dfrac{1}{3}\displaystyle\int_{0}^{2\pi}\ \mathrm{d}\theta \displaystyle\int_{0}^\pi \ \mathrm{d} \varphi \displaystyle\int_{0}^1 r^4  \sin \varphi  \ \mathrm{d}r$$
2. 球面坐标替换不属于投影法或截面法，因为积分限都是常数，不需要动态确定上下限
