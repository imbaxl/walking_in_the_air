# 1. 介绍

利用周期性条件其实直觉上不是很明显，它可能要在暗示Heine归结原理的语境下才有比较好的思路。

# 2. 例题

## 2.1. 例题1 ![[吉米多维奇连续性专题#1.7. 753. 设 $ varphi(x)$ 及 $ psi(x)$ 为定义于 $- infty < x < + infty$ 的连续周期函数，且 $ lim limits_{x to + infty} [ varphi(x) - psi(x)] = 0.$ 证明：$$ varphi(x) equiv psi(x).$$|吉米多维奇753]]

<font color="#00b050">解</font>
1. 构造数列 $x_{n}=x_{0}+nT$，$T$ 是 $f$ 已经确定的最小正周期
2. 求数列极限：$\lim\limits_{n\to \infty}x_{n}=+\infty$
3. 求函数列极限：
	- 一面（Heine归结原理）：$\lim\limits_{n\to \infty}f(x_{n})=0$
	- 二面（周期性）：$f(x_{n})=f(x_{0})$
4. 合并结果：$f(x_{0})=0$，也就是任取 $x_{0}\in (-\infty,+\infty)$ 都是 $0$.
