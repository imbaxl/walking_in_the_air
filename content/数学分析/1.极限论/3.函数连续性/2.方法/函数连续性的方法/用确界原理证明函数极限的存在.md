# 1. 用确界原理证明函数极限的存在

## 1.1. 问题背景

**754题**：证明单调有界函数的一切间断点皆为第一类间断点。

**问题转义**：第一类间断点即左右极限存在。因此，我们需要证明对于单调有界函数 $f(x)$，在任意点 $x_0$ 处，左右极限 $\lim_{x \to x_0^-} f(x)$ 和 $\lim_{x \to x_0^+} f(x)$ 都存在。

## 1.2. 关键观察

1. **注意区分**：不能使用 [[单调有界数列收敛定理]]，因为这不是数列问题，而是函数极限问题。
2. **核心工具**：证明依赖于 [[确界原理]]。
3. **单调性分类**：需要分别考虑单调递增和单调递减两种情况。

## 1.3. 证明思路

设 $f(x)$ 在区间 $I$ 上单调有界。对于任意 $x_0 \in I$，我们需要证明：
1. 左极限 $\lim_{x \to x_0^-} f(x)$ 存在
2. 右极限 $\lim_{x \to x_0^+} f(x)$ 存在

### 1.3.1. 情况一：$f(x)$ 单调递增

#### 1.3.1.1. 证明左极限存在

**步骤1：定义集合**
考虑集合 $A = \{f(x) : x < x_0, x \in I\}$。由于 $f(x)$ 有界，$A$ 是非空有界集合。

**步骤2：应用确界原理**
由确界原理，$A$ 有上确界，记 $L = \sup A$。

**步骤3：证明 $L$ 就是左极限**
我们需要证明：$\lim_{x \to x_0^-} f(x) = L$。

根据上确界的定义：
1. $L$ 是 $A$ 的上界：对于任意 $x < x_0$，有 $f(x) \leqslant L$
2. 对于任意 $\epsilon > 0$，存在 $x_\epsilon < x_0$ 使得 $f(x_\epsilon) > L - \epsilon$

由于 $f(x)$ 单调递增，当 $x_\epsilon < x < x_0$ 时：
$$L - \epsilon < f(x_\epsilon) \leqslant f(x) \leqslant L$$

因此，对于任意 $\epsilon > 0$，取 $\delta = x_0 - x_\epsilon > 0$，当 $0 < x_0 - x < \delta$ 时（即 $x_\epsilon < x < x_0$），有：
$$|f(x) - L| < \epsilon$$

这就证明了 $\lim_{x \to x_0^-} f(x) = L$ 存在。

#### 1.3.1.2. 证明右极限存在

**步骤1：定义集合**
考虑集合 $B = \{f(x) : x > x_0, x \in I\}$。由于 $f(x)$ 有界，$B$ 是非空有界集合。

**步骤2：应用确界原理**
由确界原理，$B$ 有下确界，记 $R = \inf B$。

**步骤3：证明 $R$ 就是右极限**
我们需要证明：$\lim_{x \to x_0^+} f(x) = R$。

根据下确界的定义：
1. $R$ 是 $B$ 的下界：对于任意 $x > x_0$，有 $f(x) \geqslant R$
2. 对于任意 $\epsilon > 0$，存在 $x_\epsilon > x_0$ 使得 $f(x_\epsilon) < R + \epsilon$

由于 $f(x)$ 单调递增，当 $x_0 < x < x_\epsilon$ 时：
$$R \leqslant f(x) \leqslant f(x_\epsilon) < R + \epsilon$$

因此，对于任意 $\epsilon > 0$，取 $\delta = x_\epsilon - x_0 > 0$，当 $0 < x - x_0 < \delta$ 时（即 $x_0 < x < x_\epsilon$），有：
$$|f(x) - R| < \epsilon$$

这就证明了 $\lim_{x \to x_0^+} f(x) = R$ 存在。

### 1.3.2. 情况二：$f(x)$ 单调递减

证明思路类似，只需调整不等式方向：
- 左极限：考虑集合 $\{f(x) : x < x_0\}$ 的下确界
- 右极限：考虑集合 $\{f(x) : x > x_0\}$ 的上确界

## 1.4. 综合结论

对于任意单调有界函数 $f(x)$ 和任意点 $x_0$：
1. 左极限 $\lim_{x \to x_0^-} f(x)$ 存在（等于相应集合的确界）
2. 右极限 $\lim_{x \to x_0^+} f(x)$ 存在（等于相应集合的确界）

因此，单调有界函数的一切间断点都是第一类间断点（左右极限存在）。

## 1.5. 证明方法总结

### 1.5.1. 核心技巧
1. **确界原理的应用**：将函数极限问题转化为集合的确界问题
2. **单调性的利用**：单调性确保了确界与极限的等价关系
3. **$\epsilon$-$\delta$ 语言的标准化**：将确界的定义转化为极限的定义

### 1.5.2. 证明结构
1. **定义相关集合**：根据左右极限分别定义函数值的集合
2. **应用确界原理**：证明集合有界，从而确界存在
3. **验证确界即极限**：利用单调性和确界定义证明极限存在且等于确界

### 1.5.3. 注意事项
1. **区分左右极限**：左极限对应左侧函数值的上确界（递增）或下确界（递减）
2. **区分单调性**：递增和递减情况需要分别处理
3. **边界点的处理**：对于区间端点，只需考虑单侧极限

## 1.6. 模块化应用

这个证明方法可以模块化应用于以下场景：

### 1.6.1. 应用场景1：证明单调函数的极限存在
当需要证明单调函数在某点的极限存在时，可以直接套用确界原理方法。

### 1.6.2. 应用场景2：分析函数的间断点类型
对于单调函数，可以快速判断其间断点类型：所有间断点都是第一类间断点。

### 1.6.3. 应用场景3：构造反例
理解这个证明后，可以构造只有第二类间断点的函数（这样的函数必然不是单调的）。

## 1.7. 与相关概念的连接

### 1.7.1. 与数列极限的关系
虽然不能直接使用 [[单调有界数列收敛定理]]，但思想类似：都是通过确界/极限点来证明极限存在。

### 1.7.2. 与连续性的关系
这个结论是研究 单调函数连续性 的基础。特别地，它说明了单调函数的不连续点只能是跳跃间断点。

### 1.7.3. 与实分析的关系
确界原理是实分析的基础工具之一，这个证明展示了如何将分析问题转化为集合论问题。

## 1.8. 扩展思考

1. **非单调函数的情况**：对于非单调函数，间断点可能是第二类的（如震荡间断点）
2. **无界函数的情况**：如果函数无界，间断点可能是第二类的（如无穷间断点）
3. **一致连续性**：单调连续函数在闭区间上是否一定一致连续？

## 1.9. 总结

754题的证明展示了如何将函数极限问题转化为集合的确界问题，这是实分析中的一个重要技巧。关键点包括：

1. **确界原理的核心作用**：将极限存在性问题转化为集合的确界存在性问题
2. **单调性的巧妙利用**：单调性确保了确界与函数极限的等价关系
3. **左右极限的分别处理**：需要分别考虑左极限和右极限，对应不同的集合
4. **$\epsilon$-$\delta$ 语言的标准化应用**：将确界的定义严格转化为极限的定义

这个证明不仅是754题的解答，更是一个可复用的模块，可以应用于其他需要证明单调函数极限存在性的场景。
