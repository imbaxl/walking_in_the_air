# 1. 理论

>[!dfn] 定义： 开覆盖（谢惠民符号）
>设有 $[a,b] \subset \bigcup_{\alpha} \mathcal{O}\alpha$，其中每个 $\{\mathcal{O}_{\alpha}\}$ 是开区间，则称 $\{{\mathcal{O}_{\alpha}}\}$ 是区间 $[a,b]$ 的一个开覆盖


>[!thm] 定理：覆盖定理（Heine-Borel定理）
>如果 $\{{\mathcal{O}_{\alpha}}\}$ 是区间 $[a,b]$ 上的一个开覆盖，则存在 $\{{\mathcal{O}_{\alpha}}\}$ 的一个有限子集 $\{{\mathcal{O}_{1}},\mathcal{O}_{2},\dots,\mathcal{O}_{n} \}$，它是区间 $[a,b]$ 的一个开覆盖，也就是说有 $$[a,b]\subset \bigcup_{i=1}^{n} \mathcal{O}_{i}.$$


>[!thm] 定理：覆盖定理（$\mathbb{R}^n$版本）
>$\mathbb{R}^n$ 上的点集 $S$ 是 [[紧集]] 的充分必要条件是：它是有界闭集.



>[!cor] 推论：加强形式的覆盖定理
>如果 $\{{\mathcal{O_{\alpha}}}\}$ 是区间 $[a,b]$ 的一个开覆盖，则存在一个正数 $\delta>0$，使得对区间 $[a,b]$ 中任何两个点 $x',x''$，只要 $|x'-x''|<\delta$，就存在开覆盖中的一个开区间，它覆盖 $x',x''$.（称这个数 $\delta$ 为开覆盖的 **Lebesgue 数**）

***证明：***
将得到的有限覆盖的所有元素（就是 $\mathcal{O_{1}},\mathcal{O_{2}},\dots,\mathcal{O_{n}}$）的所有端点按大小顺序排序，去掉其中重复的点，并记这个端点集为
$$A=\{{y_{0},y_{1},\dots,y_{N}}\}$$
3. 猜想加强形式覆盖定理的 **Lebesgue 数** 可能是以下形式 $$\delta=\min\{{y_{1}-y_{0},y_{2}-y_{1},\dots,y_{N}-y_{N-1}}\} $$
4. 此时从 $[a,b]$ 任取两点 $x',x''$，只有两种可能，[[f0619784890cfceef0785db2121b3952_MD5.jpeg|任意两点所处开区间的位置]]
![[f0619784890cfceef0785db2121b3952_MD5.jpeg|任意两点所处开区间位置]]
情况(a)是恰好两个点都在一个开区间中
情况(b)也是两点在一个开区间中，但中间还有一个开区间的端点，这是因为 $\delta$ 的取定是按照所有开区间的端点大小排列，因为这是有限开覆盖，开区间有相交部分，恰好两点被下一个开区间的左端点隔开，那么我们可以认为这两个点一定在上一个开区间之中，使得又或者被上一个开区间的右端点隔开，那么可以认为两个点一定在下一个开区间之中。用更简练的描述是，对于两点之间若另有 **端点集** $A$ 中的点 $\xi$，这样的点只有一个，否则超出了 $\delta$ 的范围，设该点为 $y_{i}$，与两点构成
$$x'\leqslant y_{i} \leqslant x'' $$
由于 $y_{i}$ 作为开区间的端点，该开区间没有盖住 $y_{i}$，但一定存在另一个开区间覆盖点 $y_{i}$，该开区间左端点小于 $x'$，右端点大于 $x''$，从而同时覆盖了 $x'$ 和 $x''$.




问题集
Q1：为什么要用开区间去覆盖闭区间？
A1：因为开区间有溢出的特性，能够像划分等价类一样，通过邻域传递相同的属性，用闭区间覆盖得不到很好的结论.

Q2：为什么要覆盖闭区间？
A2：因为涉及闭区间的性质很好，能够解决覆盖闭区间的问题，就能够解决很多建立在闭区间上的问题.



# 2. 应用

## 2.1. 证明零点存在定理（证明根的存在性）
?
来源：B站 [用有限覆盖定理证明根的存在性定理](https://www.bilibili.com/video/BV11N4y1Z7Jy?vd_source=a92e9f0b4b8797afe611493faaa3c133)，谢惠民（例5.2.2）
***基本思路(B站)***
1. 反证假设不存在根（即 $f(x_{0})\neq 0, x_{0} \in [a,b]$)
2. 用连续函数局部保号性推出任意一个点的邻域内函数值同号，将所有这些邻域并起来得到一个开覆盖
3. 用有限覆盖定理推出存在有限子覆盖，所有点的邻域可通过有限个邻域覆盖
4. 在有限个邻域已经覆盖了 $[a,b]$ 时，由于 $f(a)f(b)<0$，不妨假设$f(a)>0,f(b)<0$ ，那么先盖 $a$，相邻的邻域会交到 $a$ 的邻域，迫使所有相邻的邻域都与 $f(a)$ 同号，那么最后会导致 $f(a)f(b)>0$，得出矛盾
***基本思路(谢惠民例5.2.2)***
1. 与上述到第二步一致，但说明直接用覆盖定理不容易说清楚如何引出与条件 $f(a)f(b)<0$ 的矛盾，改用加强形式的覆盖定理.
2. 找到 $\delta$，使得从区间 $[a,b]$ 任取两点 $x',x''$，都有 $f(x')f(x'')>0$，从而导出矛盾.
