# 1. 唯一性

>[!thm] 定理1：唯一性
>收敛数列的极限必唯一.

***证明思路***
- 证法一（华东师大）：设数列 $x_{n}$ 的极限为 $a$，则 $\forall n>N_{1}$，$a_{n}\in U(a,\epsilon)$，若另有 $x_{n}$ 的极限为 $b$，那么 $\forall n>N_{2}$，$a_{n}\in U(b,\epsilon)$。若 $\epsilon=\dfrac{|b-a|}{2}$，那么对于邻域 $U(a,\epsilon)$，应该有无穷多项都会落入到里面，但 $U(b,\epsilon)$ 同时也有无穷多项，但是并不在邻域 $U(a,\epsilon)$ 内，因此产生矛盾。
- 证法二（陈纪修，更有技术的证法）：
	1. 几何上相当于刻画 $a$ 和 $b$ 之间的距离小于三角形两边长：$$\begin{align}|a-b|&=|a-x_{n}+x_{n}-b|\\&\leqslant |x_{n}-a|+|x_{n}-b|<\epsilon\end{align}$$
	2. 设在 $n>N_{1}$ 时有 $|x_{n}-a|<\dfrac{\epsilon}{2}$， 在 $n>N_{2}$ 时有 $|x_{n}-b|<\dfrac{\epsilon}{2}$，于是 $n>N=\max\{{ N_{1},N_{2}}\}$，有 $|a-b|<\epsilon$，
	3. 从而 $|a-b|=0$，即 $a=b$，否则若存在 $\delta$，使得 $|a-b|=\delta>0$，则令 $\epsilon=\dfrac{\delta}{2}$，使得 $|a-b|>\epsilon$，矛盾.
 
# 2. 有界性

>[!thm] 定理2：有界性
>收敛数列必有界.

***证明思路***
找界，主要分 $N$ 前后的界，$n<N$ 的界都是确定的，下标为 $1,2,\dots,N$ ，但 $n>N$ 后的界都是不确定的，所以要用极限定义确定 $\epsilon=1$ ，最后用 $\max$ 和 $\min$ 函数即可.



# 3. 保序性

>[!thm] 定理3：保序性
>设数列 $\{{x_{n}}\},\{{y_{n}}\}$  均收敛，若 $\lim\limits_{n\to \infty}x_{n}=a$，$\lim\limits_{n\to \infty}y_{n}=b$，且 $a<b$，则存在正整数 $N$ ，当 $n>N$ 时，成立$$x_{n}<y_{n}. $$

***证明思路***
保序性从唯一性的证明思路进一步延伸，即存在两个不同极限值，但是对应不同数列，那么在几何上可以这么描述：在第 $N$ 项后，两个数列都会强制落入在自己的邻域当中，而这个邻域的半径可以设在 $a$ 和 $b$ 的中间，也就是 $x_{n}<\dfrac{a+b}{2} < y_{n}$，为了产生这样的不等式，我们考虑指定 $\epsilon$，那么 $x_{n}$ 和 $y_{n}$ 至少分别满足 $x_{n}-a<\epsilon$ 和 $y_{n}-b<\epsilon$，$a+\epsilon=\dfrac{a+b}{2}$ 和 $b-\epsilon=\dfrac{a+b}{2}$，从而这样的 $\epsilon$ 只有 $\dfrac{b-a}{2}$，也就是实际上我们在控制 $\epsilon$ 从而满足  $x_{n}<\dfrac{a+b}{2} < y_{n}$
***证明***
1. 设 $\epsilon=\dfrac{b-a}{2}$，$\forall n>N_{1}:|a_{n}-a|<\epsilon$，$\forall n>N_{2}:|b_{n}-b|<\epsilon$
2. $\forall n>N=\max\{{N_{1},N_{2}}\}:x_{n}<\dfrac{a+b}{2}<y_{n}$
3. 从而 $x_{n}<y_{n}$

## 3.1. 保序性的推论（逆命题）

>[!cor] 推论4
>若 $\lim\limits_{n\to \infty}x_{n}=a,\lim\limits_{n\to \infty}y_{n}=b$，且对于 $n>N$，都有 $x_{n}\leqslant y_{n}$，那么 $$a\leqslant b. $$

***证明***
- 反证法：假设 $a>b$，由 [[#3. 保序性|保序性]] ，存在 $N$，$\forall n>N$，有 $x_{n}>y_{n}$，与条件矛盾.
- 反例，$x_{n}=\dfrac{1}{n}$，$y_{n}=\dfrac{2}{n}$，此时 $x_{n}\leqslant y_{n}$，而 $\lim\limits_{n\to \infty}x_{n}=\lim\limits_{n\to \infty}y_{n}$.

***证明思路***
取 $\epsilon=\dfrac{b-a}{2}$.


>[!cor] 推论5
>若 $\lim\limits_{n\to \infty}y_{n}=b>0$，则存在正整数 $N$ ，当 $n>N$ 时，$$y_{n}>\dfrac{b}{2}>0. $$也就是左侧数列选择了常数 $0$.


>[!cor] 推论6
>若 $\lim\limits_{n\to \infty}y_{n}=b<0$，则存在正整数 $N$ ，当 $n>N$ 时，$$y_{n}<\dfrac{b}{2}<0. $$这是右侧数列选择了常数 $0$.


# 4. 夹逼准则

>[!thm] 定理7：夹逼性
>若三个数列 $\{{x_{n}}\},\{{y_{n}}\},\{{z_{n}}\}$ 从某项开始成立
>$$x_{n}\leqslant y_{n}\leqslant z_{n},\quad n>N_{0} $$且还有 $\lim\limits_{n\to \infty}x_{n}=\lim\limits_{n\to \infty}z_{n}=a$，则 $\lim\limits_{n\to \infty}y_{n}=a.$

***证明思路***
用 $\max$ 函数使得上述条件全部成立，就有这一系列的不等式成立.
