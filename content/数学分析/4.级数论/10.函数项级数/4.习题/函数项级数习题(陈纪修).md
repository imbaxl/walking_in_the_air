## 1.1. 讨论下列函数序列在指定区间上的一致收敛性：$$\begin{align*}&\quad  (1)\ S_n(x) = e^{-nx},  \qquad \text{(i)}\ x \in (0, 1), \qquad \text{(ii)}\ x \in (1, +\infty); \\ \\&\quad  (2)\ S_n(x) = xe^{-nx},\ x \in (0, +\infty); \\ \\&\quad  (3)\ S_n(x) = \sin \frac{x}{n}, \qquad \text{(i)}\ x \in (-\infty, +\infty), \qquad \text{(ii)}\ x \in [-A, A]\ (A > 0); \\ \\&\quad  (4)\ S_n(x) = \arctan nx, \qquad \text{(i)}\ x \in (0, 1), \qquad \text{(ii)}\ x \in (1, +\infty); \\ \\&\quad  (5)\ S_n(x) = \sqrt{x^2 + \frac{1}{n^2}},\ x \in (-\infty, +\infty); \\ \\&\quad (6)\ S_n(x) = nx(1-x)^n,\ x \in [0, 1]; \end{align*} $$
?
***证明思路***
- $(1)\,S_n(x) = e^{-nx}$
	- $\text{(i)}\ x \in (0, 1)$：
		- 定理8：找出 $(0,1)$ 最大值，导数为负，从而 $n\to 0^+$ 取到最大值 $1$，从而 $d(S_{n},S)=1\not\to 0\, (n\to\infty)$，从而非一致连续
		- 定理10：选 $x_{n}=\dfrac{1}{n}$，从而非一致连续
	- $\text{(ii)}\ x \in (1, +\infty)$：
		- 定义4：$|e^{-nx}|<e^{-n}=0\ (n\to \infty)$，从而一致连续
- $(2)\,S_n(x) = xe^{-nx},\ x \in (0, +\infty)$
	- 定理10：选 $x_{n}=\dfrac{1}{n}$，趋于零，无法证明
	- 定理8：求导判断不出单调性，求导数为零的极值点为 $x=\dfrac{1}{\sqrt{ n }}$，缩小 $\dfrac{1}{n}<\dfrac{1}{\sqrt{ n }}$，代入导数，得到导数大于 $0$，从而说明极值点取极大值，求出 $d(S_{n},S)=\dfrac{1}{\sqrt{ n }}e^{-\tfrac{n}{\sqrt{ n }}}=0$，从而一致连续
- $(3)\,S_n(x) = \sin \dfrac{x}{n}$，极限函数 $S(x)=0$
	- $\text{(i)}\ x \in (-\infty, +\infty)$
		- 定理8：$d(S_{n},S)=1 \not\to 0\,(n\to \infty)$
	- $\text{(ii)}\ x \in [-A, A]\ (A > 0)$
		- 直观说明：对于 $|\sin \dfrac{x}{n}|$，负半轴的图像向上翻转，从而关于 $y$ 轴对称之下，只需要考虑正半轴。而对于 $\sin x$ 而言，最大值出现在 $\dfrac{\pi}{2}$，为了将上确界压到闭区间端点处，只要让 $\dfrac{A}{n}<\dfrac{\pi}{2}$，即 $n>\dfrac{2A}{\pi}$，$d(S_{n},S)\leqslant \dfrac{A}{n}\to 0\ (n\to \infty)$.
- $(4)\ S_n(x) = \arctan nx$，极限函数 $S(x)=\dfrac{\pi}{2}$
	- $\text{(i)}\ x \in (0, 1)$
		- 定理10：取 $x_{n}=\dfrac{1}{n}$，算出 $|S_{n}(x_{n})-S(x_{n})|=\dfrac{\pi}{4}\not\to0\ (n\to \infty)$
		- 定理8：$x=0$ 时能取到上确界 $|\arctan 0 - \dfrac{\pi}{2}|$，从而 $d(S_{n},S)=\dfrac{\pi}{2}\not\to 0 \ (n\to \infty)$
	- $\text{(ii)}\ x \in (1, +\infty)$
		- 定理8：当 $x=1$ 时，$\sup\limits_{x\in D}|\arctan nx - \dfrac{\pi}{2}|=\dfrac{\pi}{2}-\arctan n\to 0 \ (n\to \infty)$
- $(5)\ S_n(x) = \sqrt{x^2 + \dfrac{1}{n^2}},\ x \in (-\infty, +\infty)$
	- 定理10：取 $x_{n}=\dfrac{1}{n}$，数列极限为 $0$，无法证明不收敛
	- 定理8：求导，算出 $x=0$ 是极值点，代入 $x=\dfrac{1}{n}$ 会使得导数为负，从而确认 $x=0$ 是极大值点，因此代入作为上确界，得到 $d(S_{n},S)=\dfrac{1}{n}\to 0\ (n\to \infty)$
- $(6)\ S_n(x) = nx(1-x)^n,\ x \in [0, 1]$
	- 定理10：取 $x_{n}=\dfrac{1}{n}$，数列极限为 $e^{-1}\not\to 0\ (n\to \infty)$




## 1.2. 讨论下列函数序列在指定区间上的一致收敛性：$$\begin{align*}&\quad (7)\ S_n(x) = \frac{x}{n} \ln \frac{x}{n}, \qquad \text{(i)}\ x \in (0, 1), \qquad \text{(ii)}\ x \in (1, +\infty); \\ \\&\quad  (8)\ S_n(x) = \frac{x^n}{1+x^n},  \qquad\text{(i)}\ x \in (0, 1), \qquad\text{(ii)}\ x \in (1, +\infty); \\ \\&\quad (9)\ S_n(x) = (\sin x)^n,\ x \in [0, \pi]; \\ \\&\quad  (10)\ S_n(x) = (\sin x)^{\tfrac{1}{n}}, \\&\qquad \text{(i)}\ x \in [0, \pi], \qquad \text{(ii)}\ x \in [\delta, \pi-\delta]\ (\delta > 0); \\ \\&\quad (11)\ S_n(x) = \left(1+\frac{x}{n}\right)^n, \\&\qquad \text{(i)}\ x \in (-\infty, +\infty), \qquad\text{(ii)}\ x \in [-A, A]\ (A > 0); \\ \\&\quad  (12)\ S_n(x) = n(\sqrt{x+\frac{1}{n}}-\sqrt{x}),  \\ & \qquad\text{(i)}\ x \in (0, +\infty), \qquad\text{(ii)}\ x \in [\delta, +\infty),\ \delta > 0.\end{align*} $$
?
***证明思路***
- $(7)\ S_n(x) = \dfrac{x}{n} \ln \dfrac{x}{n}$
	- 计算极限函数，不定式，考虑变形，$=\dfrac{x}{n}(\ln x - \ln n)=0\ (n\to \infty)$，$S(x)=0$
	- $\text{(i)}\ x \in (0, 1)$
		- $(\dfrac{x}{n}\ln \dfrac{x}{n})'=\dfrac{1}{n}(\ln \dfrac{x}{n} +1)$，括号内式子 $\ln \dfrac{ex}{n}<\ln \dfrac{e}{n}<\ln 1=0\ (n>2)$ 成立，从而构造的距离函数单调递减，并且导数为零算出极值点在 $x=\dfrac{n}{e}$，可以看出 $n>3$ 时不在 $(0,1)$ 内，因此极值点应当在端点处.
		- 考虑在 $x=1$ 的端点取上确界，$|S_{n}(0+ )-S(0+)|=0$，$|S_{n}(1-)-S(1-)|=\dfrac{\ln n }{n}\to 0 \ (n\to \infty)$
		- 从而 $d(S_{n},S)=\dfrac{\ln n}{n}\to 0\ (n\to \infty)$ 
	- $\text{(ii)}\ x \in (1, +\infty)$
		- 定理10：取 $x_{n}=\dfrac{1}{n^2}$，数列极限为 $n\ln n \to \infty \not\to 0\ (n\to \infty)$
- $(8)\ S_n(x) = \dfrac{x^n}{1+x^n}$
	- 极限函数  $S(x)=1$
	- $\text{(i)}\ x \in (0, 1)$
		- 定理8：求导判断 $x=0$ 是极值点，单调递减，从而 $x=0$ 是极大值点，代入 $|S_{n}(0)-S(0)|=\dfrac{1}{1+0^n}=1$ ，求极限也是 $1$，无法证明一致收敛
		- 定理10：取 $x_{n}=\dfrac{1}{n}$，$\lim\limits_{n\to \infty}(S_{n}(\dfrac{1}{n})-S(\dfrac{1}{n}))=1>0$，从而不一致收敛
	- $\text{(ii)}\ x \in (1, +\infty)$
		- 定理10：取 $x_{n}=n$，$\lim\limits_{n\to \infty}(S_{n}(\dfrac{1}{n})-S(\dfrac{1}{n}))=1>0$，从而不一致收敛
- $(9)\ S_n(x) = (\sin x)^n,\ x \in [0, \pi]$
	- 极限函数为 $S(x)=\begin{cases}0& x\in[0,\pi],x\neq\dfrac{\pi}{2}  \\1&x=\dfrac{\pi}{2} \end{cases}$
	- 找出最大距离 $d(S_{n},S)$，直观看出 $x\to \dfrac{\pi}{2}$ 时，$d(S_{n},S)=1$，因此考虑用定理10
	- 定理10：与以往一整块区间不同，$x=\dfrac{\pi}{2}$ 极限函数不同，单单讨论这一点的距离为 $0$，为了用上定理10，构造 $x_{n}$ ，由 [[中值定理(介值定理)]]，在 $[0,1]$ 之间找到一组数列，使得这组数列都不会取到 $\dfrac{\pi}{2}$，只要 $\sin x$ 取不到 $1$，因此 $x_{n}=1-\dfrac{1}{n}$ 满足要求，而 $1-\dfrac{1}{n}\in [0,1]$ （值域），总存在 $x\in [0,\pi]$ ，随着 $n$ 增大，$1-\dfrac{1}{n}$ 都有对应的 $x_{n}$ 满足，从而这样的 $x_{n}$ 存在性得到保证. 由定理10，数列极限 $(1-\dfrac{1}{n})^n\not\to 0\ (n\to \infty)$ 
- $(10)\ S_n(x) = (\sin x)^{\tfrac{1}{n}}$
	- 极限函数为 $S(x)=\begin{cases}0& x=0,\pi  \\1&0<x<\pi \end{cases}$
	- $\text{(i)}\ x \in [0, \pi]$
		- 直观分析，当 $x\to 0+$ 或 $x\to \pi -$， $|S_{n}(x)-S(x)|=1$，考虑用定理10
		- 定理10：构造 $x_{n}$，若代入 $x_{n}=\dfrac{1}{n}$，得不到不一致收敛，参考第九题的构造方法
		- <font color="#ff0000">构造反例数列</font>：
			1. 取 $x_{n}\in (0,\pi)$，使得 $\sin x_{n}=\dfrac{1}{2^n}$
			2. 存在性论证：由于 $\sin x$ 是连续函数，由 [[中值定理(介值定理)]]，在闭区间 $[0,\pi]$ 存在 $x_{n}$ 使得 $0\leqslant \sin x_{n}\leqslant 1$，剔除端点 $x=0,\pi$ 后，$\sin x_{n}$ 无法达到最大值和最小值，但 $0<\sin x_{n}<1$，$\sin x_{n}=\dfrac{1}{2^n}$ 正好满足要求，此时在计算数列极限的距离时就不需要 $x_{n}$ 的显式形式.
