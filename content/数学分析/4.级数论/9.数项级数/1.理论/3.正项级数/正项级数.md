>[!dfn] 定义1（正项级数）
>如果级数 $\displaystyle\sum\limits_{n=1}^{\infty}x_{n}$ 的各项都是非负实数，即 $$x_{n}\geqslant 0,\quad n=1,2,\cdots, $$则称此级数为 **正项级数**.

根据定义可以看出部分和数列 $S_{n}$ 是单调递增，因此有如下定理判断收敛

>[!thm] 定理2
>正项级数收敛的充分必要条件是它的部分和数列有上界.

>[!prp] 例题3
>判断级数 $\displaystyle\sum\limits_{n=2}^{\infty}\Big[ \dfrac{1}{\sqrt[n]{ n }} \ln \dfrac{n^2}{(n-1)(n+1)}  \Big]$ 是否收敛.

<font color="#00b050">解</font>
1. 可以看出是正项级数，因为没有负数的可能
2. 由 **定理2**，接下来只需要判断是否有界，考虑放大
	- 部分和 $\leqslant \displaystyle\sum\limits_{n=2}^{n+1}(\ln \dfrac{k^2}{(k-1)(k+1)})$
	- $= \displaystyle\sum\limits_{n=2}^{n+1}(\ln \dfrac{k}{k-1}-\ln \dfrac{k+1}{k})$
	- $=\ln2 - \ln \dfrac{n+2}{n+1}<\ln2$
3. 结论成立.

>[!thm] 定理4（比较判别法）
>设 $\sum\limits_{n = 1}^{\infty} x_{n}$ 与 $\sum\limits_{n = 1}^{\infty} y_{n}$ 是两个正项级数, 若存在常数 $A > 0$, 使得
$$ x_{n} \leq A y_{n}, \quad n = 1, 2, \cdots, $$
则
(1) 当 $\sum\limits_{n = 1}^{\infty} y_{n}$ 收敛时, $\sum\limits_{n = 1}^{\infty} x_{n}$ 也收敛;
>
(2) 当 $\sum\limits_{n = 1}^{\infty} x_{n}$ 发散时, $\sum\limits_{n = 1}^{\infty} y_{n}$ 也发散.



>[!prp] 例题5.1
>判断正项级数 $\displaystyle\sum\limits_{n=1}^{\infty}\dfrac{n+3}{2n^3-n}$ 的敛散性.

<font color="#00b050">解</font>
- 考虑 **定理4** 的比较判别法
- 对 $\dfrac{n+3}{2n^3-n}$ 进行放大，目标形式应该类似于 $\dfrac{1}{n^2}$
- 尝试1：$\dfrac{n+3}{2n^3-n}< \dfrac{n}{2n^3}$ 是错误示范
- 尝试2：$\dfrac{n+3}{2n^3-n}< \dfrac{n}{n^3}$，以带有猜想的手法断言，所以还要检验
	- $n=1$ 有 $\dfrac{4}{1}<1$，不成立
	- $n=2$ 有 $\dfrac{5}{14}<\dfrac{1}{4}$，不成立
	- $n=3$ 有 $\dfrac{6}{51}<\dfrac{1}{9}$，不成立
	- $n=4$ 有 $\dfrac{7}{124}<\dfrac{1}{16}=\dfrac{7}{112}$，成立
	- 分母放大一倍在 $n=4$ 后总是成立不等式


>[!prp] 例题5.2
>判断正项级数 $\displaystyle\sum\limits_{n=1}^{\infty}\sin \dfrac{\pi}{n}$ 的敛散性.

<font color="#00b050">解</font>
- 继续应用定理4完成判断
- 如果用 $\sin x \leqslant x (x\in [0,\dfrac{\pi}{2}])$，右侧是发散，但没法判断左侧 
- 使用 $\sin x\geqslant \dfrac{2}{\pi}x$
	- $\sin \dfrac{\pi}{n}\geqslant \dfrac{2}{\pi}\dfrac{\pi}{n}=\dfrac{2}{n}$
	- 由定理4，$\displaystyle\sum\limits_{n=1}^{\infty}\dfrac{2}{n}$ 发散，从而 $\displaystyle\sum\limits_{n=1}^{\infty}\sin \dfrac{\pi}{n}$ 发散

>[!cnj] 小结
>上述两道例题明显展示出 **定理4** 一般需要配合常见的放缩技巧和常见的不等式来辅助验证敛散性，而 **例题5.2** 更进一步说明了配备充足的不等式储备，才能高效解题.

由定理4移项取极限即可得到定理4的极限形式：

>[!thm] 定理6（比较判别法的极限形式）
>设 $\sum\limits_{n = 1}^{\infty} x_{n}$ 与 $\sum\limits_{n = 1}^{\infty} y_{n}$ 是两个正项级数, 且 $\lim\limits_{n \to \infty} \dfrac{x_{n}}{y_{n}} = l$ ($0 \leq l \leq +\infty$), 则
(1) 若 $0 \leq l < +\infty$, 则当 $\sum\limits_{n = 1}^{\infty} y_{n}$ 收敛时, $\sum\limits_{n = 1}^{\infty} x_{n}$ 也收敛;
(2) 若 $0 < l \leq +\infty$, 则当 $\sum\limits_{n = 1}^{\infty} y_{n}$ 发散时, $\sum\limits_{n = 1}^{\infty} x_{n}$ 也发散.
所以当 $0 < l < +\infty$ 时, $\sum\limits_{n = 1}^{\infty} x_{n}$ 与 $\sum\limits_{n = 1}^{\infty} y_{n}$ 同时收敛或同时发散.
