>[!dfn] 定义1（无穷数项级数）
>设 $x_{1}$, $x_{2}$, $\cdots$, $x_{n}$, $\cdots$ 是无穷可列个实数，我们称它们的“和” 
$$ x_{1} + x_{2} + \cdots + x_{n} + \cdots $$ 
为 **无穷数项级数** (简称**级数**)，记为 $\sum\limits_{n=1}^{\infty} x_{n}$，其中 $x_{n}$ 称为级数的 **通项** 或一般项.



>[!dfn] 定义2（无穷级数收敛，发散）
>如果部分和数列 $\{S_{n}\}$ 收敛于有限数 $S$，则称无穷级数 $\sum\limits_{n=1}^{\infty} x_{n}$ 收敛，且称它的和为 $S$，记为
$$ S = \sum\limits_{n=1}^{\infty} x_{n} $$
如果部分和数列 $\{S_{n}\}$ 发散，则称无穷级数 $\sum\limits_{n=1}^{\infty} x_{n}$ 发散.



>[!prp] 例题3.1（几何级数）
>设 $|q| < 1$，则几何级数 (即等比级数)$$
\sum\limits_{n = 1}^{\infty} q^{n - 1} = 1 + q + q^{2} + \cdots + q^{n} + \cdots$$
是收敛的. 这是因为它的部分和数列的通项为$$S_{n} = \sum\limits_{k = 1}^{n} q^{k - 1} = \frac{1 - q^{n}}{1 - q},$$
显然，$$\lim\limits_{n \to \infty} S_{n} = \frac{1}{1 - q}.$$

- 按定义验证，由于收敛到固定数 $\dfrac{1}{1-q}$，从而级数收敛.

>[!prp] 例题3.2
>级数 $$\sum\limits_{n = 1}^{\infty} (-1)^{n - 1} = 1 - 1 + 1 - \cdots + (-1)^{n - 1} + \cdots$$
是发散的. 这是因为它的部分和数列的通项为$$S_{n} = \begin{cases} 0, & n \text{ 为偶数}, \\ 1, & n \text{ 为奇数.}\end{cases}$$
显然 $\{S_{n}\}$ 是发散的.

- 按定义验证：由于部分和数列发散，从而级数发散.

>[!prp] 例题3.3
>$p$ 级数 $$\sum\limits_ { n = 1 } ^ { \infty } \dfrac { 1 } { n ^ { p } } = 1 + \dfrac { 1 } { 2 ^ { p } } + \dfrac { 1 } { 3 ^ { p } } + \cdots + \dfrac { 1 } { n ^ { p } } + \cdots( p > 0 )$$ 当 $p > 1$ 时收敛；当 $0 < p \leqslant 1$ 时发散到正无穷大.
>当 $p=1$ 时称为 **调和级数**. 

- 证明参考 [[数列极限习题(谢惠民)#2.1. （例6）设 $S_{n}$ $=$ $1$ $+$ $ dfrac{1}{2}$ $+$ $ dfrac{1}{3}$ $+$ $ cdots$ $+$ $ dfrac{1}{n}$，$n$ $ in$ $N_{+}$，证明数列 $ { S_{n} }$ 发散.|谢惠民数列极限，例6]]；

>[!thm] 定理4（级数收敛的必要条件）
>设级数$\sum\limits_{n = 1}^{\infty} x_{n}$ 收敛，则其通项所构成的数列  $\{x_{n}\}$ 是无穷小量，即
$$\lim\limits_{n \to \infty} x_{n} = 0.$$

- 因为是找到大前提，所以用法是找到发散的通项，就可以说明级数发散.
- 证明思路：利用形式 $a_{n}=S_{n}-S_{n-1}$，两侧求极限，由于增减一项不改变极限值，从而使得 $\lim\limits_{n\to \infty}a_{n}=0$
- 反例：调和级数


>[!thm] 定理5（级数的线性性）
>设 $\sum\limits_{n=1}^{\infty}$ $a_{n}$ $=$ $A$，$\sum\limits_{n=1}^{\infty}$ $b_{n}$ $=$ $B$，$\alpha$，$\beta$是两个常数，则$$\sum_{n=1}^{\infty}(\alpha a_{n}+ \beta b_{n}) = \alpha A + \beta B.$$


>[!prp] 例题6
>求级数 $\displaystyle\sum\limits_{n=1}^{\infty}\dfrac{4^{n+1}-3\cdot 2^n}{5^n}$ 的值.

***问题***
1. $\displaystyle\sum\limits_{n=1}^{\infty}q^{n-1}$ 和 $\displaystyle\sum\limits_{n=0}^{\infty}q^n$ 是等价的吗，有何区别？
- $n=0$ 时要求表达式是 $q^n$ ，但 $n=1$ 时要求表达式是 $q^{n-1}$，满足条件的话，这两种下标所表达的级数是一样的，也就是两者都是在表达：$$1+q+q^2+\cdots+q^{n-1} $$
****
***计算***
1. 原式=$\displaystyle\sum\limits_{n=1}^{\infty}(4(\dfrac{4}{5})^n-3(\dfrac{2}{5})^n)$，注意到下标 $n=1$，对应的几何级数幂次是 $n-1$，因此提出一次因子就可以转换成几何级数
2. 得到 $\dfrac{16}{5}\displaystyle\sum\limits_{n=1}^{\infty}(\dfrac{4}{5})^{n-1}-\dfrac{6}{5}\displaystyle\sum\limits_{n=1}^{\infty}(\dfrac{2}{5})^{n-1}$，直接计算得到 $14$.
3. 或者直接补项成 $n=0$，也就是化成 $4\left( \displaystyle\sum\limits_{n=0}^{\infty} (\dfrac{4}{5})^n-1\right)-3\left( \displaystyle\sum\limits_{n=1}^{\infty}(\dfrac{2}{5})^n-1 \right)$



>[!thm] 定理7（级数收敛的充分条件）
>设级数 $\displaystyle\sum\limits_{n=1}^{\infty}x_{n}$ 收敛，则在它的求和表达式中任意添加括号后所得的数仍然收敛，且其和不变.

***证明思路***
1. 比如选取 $x_{1},x_{2},\cdots,x_{20}$，其部分和数列为 $S_{1},\cdots,S_{n}$
2. 将 $U_{1}=x_{1},x_{2},x_{3},x_{4}$，每4个一组，那么共有 $U_{1},U_{2},U_{3},U_{4},U_{5}$
3. 可以发现部分和数列为 $S_{1},S_{2},S_{3},U_{1},S_{5},\cdots,U_{2},\cdots,S_{19},U_{5}$，这说明了 $U_{k}$ 是 $S_{n}$ 的子列，因此 $U_{k}$ （任意加括号的部分和）也收敛
****
***应用***
- 收敛的级数满足加法结合律
****
***反例***
- 发散级数：$\displaystyle\sum\limits_{n=1}^{\infty}(-1)^{n-1}$，$(1-1)+(1-1)+\cdots$，对括号级数求部分和是收敛到 $0$.
- 发散级数：$\displaystyle\sum\limits_{n=1}^{\infty}(-1)^{n-1}$，$1+(-1+1)+(-1+1)+\cdots$，对括号级数求部分和是收敛到 $1$.



>[!prp] 例题8.1
>求二进制无限循环小数 $(110.110110\cdots )_{2}$ 的值.

***计算***
- 拆解成 $2^k$ 的求和，$0$ 的位置省略，那么 $$(110.110110\cdots )_{2}=2^2+2^1+2^{-1}+2^{-2}+2^{-4}+2^{-5}+\cdots$$
- 考虑 **定理7**，如果给上述每两项加括号，就可以得到一个新的部分和 $S_{2n}$，但是需要表达出具体式子
- 可以看出幂次是等差数列的规律，就可以得到 $\displaystyle\sum\limits_{n=1}^{\infty}(\dfrac{1}{2^{3n-5}}+\dfrac{1}{2^{3n-4}})$，其部分和是 $\displaystyle\sum\limits_{k=1}^{n}(\dfrac{1}{2^{3k-5}}+\dfrac{1}{2^{3k-4}})$，
- 此处可提出因子 $2^{3k-3}$，也可以分开计算两个级数
- 利用 **定理5**，凑出几何级数，算出 $\dfrac{48}{7}$，但这只是 $\{{S_{2n}}\}$ 的极限 ，这是偶子列的极限，若奇子列的极限也一样，那么原数列的极限也一样，参考 [[数列极限习题(陈纪修)#2.4. 设$ lim limits_{n to infty}x_{2n}= lim limits_{n to infty}x_{2n+1}=a$，证明：$ lim limits_{n to infty}x_{n}=a$|陈纪修例题]].
- 于是 $S_{2n+1}=S_{2n}+\dfrac{1}{2^{3n-2}}$，求极限则两侧极限相等，从而原部分和极限为 $\dfrac{48}{7}$.

>[!prp] 例题8.3
>计算级数 $\displaystyle\sum\limits_{n=1}^{\infty}\dfrac{2n-1}{2^n}$.

***计算思路***
- 此题带来很明显的感受是目前能计算的级数非常有限，只有满足几何级数的形式才能计算，而此题要化为几何级数至少一眼看去并不明显
- 此题用到的技巧
	- 调整 $\sum\limits$ 的首尾项，使其可以在相同的 $\sum\limits$ 下合并运算
	- 利用 $S_{n}=S_{2n}-S_{n}$




>[!prp] 例8.4*
>计算级数 $\displaystyle\sum\limits_{n=1}^{\infty}\arctan \dfrac{1}{2n^2}$.

***计算思路***
- 要利用公式 $\arctan x-\arctan y = \arctan \dfrac{x-y}{1-xy}$
	- 这里是从左侧猜 $x=\dfrac{1}{2n-1},y=\dfrac{1}{2n+1}$
- 得到 $\displaystyle\sum\limits_{n=1}^{\infty}\arctan \dfrac{1}{2n^2}=\displaystyle\sum\limits_{n=1}^{\infty} \arctan 1 - \arctan \dfrac{1}{2n+1}=\dfrac{\pi}{4}.$
