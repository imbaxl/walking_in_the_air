>[!dfn] 定义1（无穷数项级数）
>设 $x_{1}$, $x_{2}$, $\cdots$, $x_{n}$, $\cdots$ 是无穷可列个实数，我们称它们的“和” 
$$ x_{1} + x_{2} + \cdots + x_{n} + \cdots $$ 
为 **无穷数项级数** (简称**级数**)，记为 $\sum\limits_{n=1}^{\infty} x_{n}$，其中 $x_{n}$ 称为级数的 **通项** 或一般项.



>[!dfn] 定义2（无穷级数收敛，发散）
>如果部分和数列 $\{S_{n}\}$ 收敛于有限数 $S$，则称无穷级数 $\sum\limits_{n=1}^{\infty} x_{n}$ 收敛，且称它的和为 $S$，记为
$$ S = \sum\limits_{n=1}^{\infty} x_{n} $$
如果部分和数列 $\{S_{n}\}$ 发散，则称无穷级数 $\sum\limits_{n=1}^{\infty} x_{n}$ 发散.



>[!prp] 例题3.1（几何级数）
>设 $|q| < 1$，则几何级数 (即等比级数)$$
\sum\limits_{n = 1}^{\infty} q^{n - 1} = 1 + q + q^{2} + \cdots + q^{n} + \cdots$$
是收敛的. 这是因为它的部分和数列的通项为$$S_{n} = \sum\limits_{k = 1}^{n} q^{k - 1} = \frac{1 - q^{n}}{1 - q},$$
显然，$$\lim\limits_{n \to \infty} S_{n} = \frac{1}{1 - q}.$$

- 按定义验证，由于收敛到固定数 $\dfrac{1}{1-q}$，从而级数收敛.

>[!prp] 例题3.2
>级数 $$\sum\limits_{n = 1}^{\infty} (-1)^{n - 1} = 1 - 1 + 1 - \cdots + (-1)^{n - 1} + \cdots$$
是发散的. 这是因为它的部分和数列的通项为$$S_{n} = \begin{cases} 0, & n \text{ 为偶数}, \\ 1, & n \text{ 为奇数.}\end{cases}$$
显然 $\{S_{n}\}$ 是发散的.

- 按定义验证：由于部分和数列发散，从而级数发散.

>[!prp] 例题3.3
>$p$ 级数 $$\sum\limits_ { n = 1 } ^ { \infty } \dfrac { 1 } { n ^ { p } } = 1 + \dfrac { 1 } { 2 ^ { p } } + \dfrac { 1 } { 3 ^ { p } } + \cdots + \dfrac { 1 } { n ^ { p } } + \cdots( p > 0 )$$ 当 $p > 1$ 时收敛；当 $0 < p \leqslant 1$ 时发散到正无穷大.
>当 $p=1$ 时称为 **调和级数**. 

- 证明参考 [[数列极限习题(谢惠民)#2.1. （例6）设 $S_{n}$ $=$ $1$ $+$ $ dfrac{1}{2}$ $+$ $ dfrac{1}{3}$ $+$ $ cdots$ $+$ $ dfrac{1}{n}$，$n$ $ in$ $N_{+}$，证明数列 $ { S_{n} }$ 发散.|谢惠民数列极限，例6]]；

>[!thm] 定理4（级数收敛的必要条件）
>设级数$\sum\limits_{n = 1}^{\infty} x_{n}$ 收敛，则其通项所构成的数列  $\{x_{n}\}$ 是无穷小量，即
$$\lim\limits_{n \to \infty} x_{n} = 0.$$

- 因为是找到大前提，所以用法是找到发散的通项，就可以说明级数发散.
- 证明思路：利用形式 $a_{n}=S_{n}-S_{n-1}$，两侧求极限，由于增减一项不改变极限值，从而使得 $\lim\limits_{n\to \infty}a_{n}=0$
- 反例：调和级数


>[!thm] 定理5（级数的线性性）
>设 $\sum\limits_{n=1}^{\infty}$ $a_{n}$ $=$ $A$，$\sum\limits_{n=1}^{\infty}$ $b_{n}$ $=$ $B$，$\alpha$，$\beta$是两个常数，则$$\sum_{n=1}^{\infty}(\alpha a_{n}+ \beta b_{n}) = \alpha A + \beta B.$$
