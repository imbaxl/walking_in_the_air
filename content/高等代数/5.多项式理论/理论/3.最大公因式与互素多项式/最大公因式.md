# 1. 定义

>[!dfn] 定义1：最大公因式 
>设 $d(x)$ 是 $f(x),g(x)$ 的公因式，若对 $f(x),g(x)$ 的任一公因式 $h(x)$ ，都有 $h(x)|d(x)$ ，则 $d(x)$ 是 $f(x),g(x)$ 的最大公因式，记$$ d(x)=(f(x),g(x)).$$



>[!thm] 定理2：最大公因式的充分条件
>$d(x)$ 是 $f(x)$ 和 $g(x)$ 的最大公因式，则必存在 $u(x),v(x)$ 使得$$ f(x)u(x)+g(x)v(x)=d(x).$$


# 2. 理解

初步的推导是通过 [[辗转相除法]] 得到余式为零的公因式，具体来说就是当我们得到如下式子
$$r_{s-1}(x)=q_{s+1}(x)r_{s}(x)+0 $$
此时，$r_{s}(x)$ 就是 $r_{s}(x)$ 与 $r_{s-1}(x)$ 的最大公因式，再通过如下引理，把最大公因式传递回 $f(x)$ 和 $g(x)$ 即可.

>[!lem] 引理3
>如果有等式
>$$f(x)=q(x)g(x)+r(x) $$
>成立，那么 $f(x)$，$g(x)$ 和 $g(x)$，$r(x)$ 有相同的公因式.
