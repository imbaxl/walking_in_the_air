# 1. 引言
定义 $k$ 重因式是北大版教材的推导路径。

# 2. 定义 

>[!dfn] 定义1： $k$ 重因式
>若不可约多项式 $p(x)$ 满足 $p^{k}(x)\mid f(x)$，而且 $p^{k+1}(x)\nmid f(x)$，则 $p(x)$ 是 多项式 $f(x)$ 的 $k$ 重因式.
>

# 3. 定理及推论

>[!thm] 定理2
>如果不可约多项式 $p(x)$ 是 $f(x)$ 的 $k$ 重因式 $(k\geqslant1)$，那么它是形式导数 $f'(x)$ 的 $k-1$ 重因式.


>[!cor] 推论3
>如果不可约多项式 $p(x)$ 是 $f(x)$ 的 $k$ 重因式 $(k\geqslant 1)$，那么 $p(x)$ 是 $f(x),f'(x),\cdots,f^{(k-1)}(x)$ 的因式，但不是 $f^{(k)}(x)$ 的因式.

***证明***
考虑用数学归纳法
1. $k=1$ ：用 定义1 和 定理2 即可得证，也就是形式导数得到的重数是0，故不是因式
2. 假设 $k=m$ 成立：$p(x)$ 是 $f(x),f'(x),\cdots,f^{(k-1)}(x)$ 的因式，但不是 $f^{(k)}(x)$ 的因式.
3. 证 $k=m+1$ 也成立：先用定理2得到 $f'(x)$，$p(x)$  是 $f'(x)$ 的 $m$ 重因式，根据归纳假设，$p(x)$ 是 $f'(x),f^{''}(x),\cdots,f^m(x)$ 的因式，但不是 $f^{m+1}(x)$ 的因式.


>[!cor] 推论4
>不可约多项式 $p(x)$ 是 $f(x)$ 的重因式的充分必要条件是 $p(x)$ 是 $f(x)$ 与 $f'(x)$ 的公因式.  

***证明***
必要性：推论3即得
充分性：考虑反证法。假设 $p(x)$ 是 $f(x)$ 的单因式，那么 $f(x)=p(x)g(x)$，其中 $p(x)\nmid g(x)$，而 $f'(x)=p'(x)g(x)+p(x)g'(x)$，由于 $p(x)\nmid p'(x)$，$p(x)$ 不是 $f'(x)$ 的因式，矛盾，故原假设成立。
