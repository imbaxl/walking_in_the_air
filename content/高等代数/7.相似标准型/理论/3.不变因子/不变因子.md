引言：[[矩阵的法式]] 说明了任何一个 $\lambda-$矩阵都相抵于一个对角 $\lambda-$矩阵，也就是任给 $A(\lambda),B(\lambda)$，如果他们相抵于同一个法式，那么 $A(\lambda)$ 与 $B(\lambda)$ 就是相抵的；反过来，能否说明如果 $A(\lambda)$ 和 $B(\lambda)$ 的法式不相同，则他们一定不相抵呢？
通过引入行列式因子等概念，帮助这个猜想的证明.

| 排序      | 内容                            |

| 定义1     | 行列式因子                         |
| 引理2     | 行列式因子也能一个整除一个                 |

| 定理4     | 相抵 $\lambda-$矩阵有相同的行列式因子和不变因子 |
| 推论5     | 能“对齐”不变因子的法式形状                |
| 推论6     | 解答了引言，相抵当且仅当有相同的法式            |
| 推论7     | 法式与初等变换的选取无关                  |
| **定理8** | 数字矩阵相似当且仅当有相同的行列式因子或不变因子      |
| 推论9     | 小数域相似，大数域也相似                  |

>[!dfn] 定义1（行列式因子）
>设 $A(\lambda)$ 是 $n$ 阶 $\lambda$-矩阵，$k$ 是小于等于 $n$ 的正整数. 如果 $A(\lambda)$ 有一个 $k$ 阶子式不为零，则定义 $A(\lambda)$ 的 $k$ 阶行列式因子 $D_{k}(\lambda)$ 为 $A(\lambda)$ 的所有 $k$ 阶子式的最大公因式(首一多项式). 如果 $A(\lambda)$ 的所有 $k$ 阶子式都等于零，则定义 $A(\lambda)$ 的 $k$ 阶行列式因子 $D_{k}(\lambda)$ 为零.



>[!lem] 引理2
>设 $D_{1}(\lambda)$, $D_{2}(\lambda)$, $\cdots$, $D_{r}(\lambda)$ 是 $A(\lambda)$ 的非零行列式因子，则
$$ D_{i}(\lambda) \mid D_{i+1}(\lambda), \quad i = 1, 2, \cdots, r - 1. $$





>[!dfn] 定义3（不变因子）
>设 $D_{1}(\lambda)$, $D_{2}(\lambda)$, $\cdots$, $D_{r}(\lambda)$ 是 $\lambda-$矩阵 $A(\lambda)$ 的非零行列式因子，则 $g_{1}(\lambda) = D_{1}(\lambda)$, $g_{2}(\lambda) = D_{2}(\lambda)/D_{1}(\lambda)$, $\cdots$, $g_{r}(\lambda) = D_{r}(\lambda)/D_{r-1}(\lambda)$ 称为 $A(\lambda)$ 的**不变因子**.





>[!thm] 定理4
>相抵的 $\lambda-$矩阵有相同的行列式因子，从而有相同的不变因子.



>[!cor] 推论5
>设 $n$ 阶 $\lambda-$矩阵 $A(\lambda)$ 的法式为
$$ A = \operatorname{diag}\left\{d_{1}(\lambda), d_{2}(\lambda), \cdots, d_{r}(\lambda); 0, \cdots, 0\right\} $$
其中 $d_{i}(\lambda)$ 是非零首一多项式且 $d_{i}(\lambda) \mid d_{i+1}(\lambda)$ ($i = 1$, $2$, $\cdots$, $r - 1$)，则 $A(\lambda)$ 的不变因子为 $d_{1}(\lambda)$, $d_{2}(\lambda)$, $\cdots$, $d_{r}(\lambda)$. 特别，法式和不变因子之间相互唯一确定.




>[!cor] 推论6
>设 $A(\lambda)$, $B(\lambda)$ 为 $n$ 阶 $\lambda-$矩阵，则 $A(\lambda)$ 与 $B(\lambda)$ 相抵当且仅当它们有相同的法式.



>[!cor] 推论7
>$n$ 阶 $\lambda-$矩阵 $A(\lambda)$ 的法式与初等变换的选取无关.



>[!thm] 定理8
>数域 $\mathbb{K}$ 上 $n$ 阶矩阵 $A$ 与 $B$ 相似的充分必要条件是它们的特征矩阵 $\lambda I - A$ 与 $\lambda I - B$ 具有相同的行列式因子或不变因子.



>[!cor] 推论9
>设 $\mathbb{F} \subseteq \mathbb{K}$ 是两个数域，$A$, $B$ 是 $\mathbb{F}$ 上的两个矩阵，则 $A$ 与 $B$ 在 $F$ 上相似的充分必要条件是它们在 $\mathbb{K}$ 上相似.
