二次型可以说是某个向量与自己的内积，但是结果可以是负定的，不受内积空间的正定性约束，所以这两个知识点：[[二次型]] 和 [[内积空间]] 是有重合的关系。

相似的地方就在于形式： $$二次型:f(x)=x'Ax=(x,x):内积$$
不同之处：


|        | 二次型         | 内积            | 辛几何  |
| ------ | ----------- | ------------- | ---- |
| 约束     | 正定负定都可以     | 只能是正定         |      |
| 目的     | 分析函数        | 定义空间          |      |
| 合同变换目标 | 用对角元正负号数量分类 | 单位阵，标准正交基     |      |
| 惯性指数   | 可变，所以需要强调   | 恒为 $n$，所以不用说明 |      |
| 双线性型视角 | 对称性         | 对称性、正定性       | 反对称性 |
