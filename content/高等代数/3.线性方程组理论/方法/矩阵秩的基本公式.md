# 1. 矩阵数乘的秩

>[!prp] 命题1：若 $k\neq 0$，那么 $\text{r}(kA)=\text{r}(A)$

# 2. 矩阵乘积的秩不等式

>[!prp] 命题：$\text{r}(AB)\leqslant \min \{{ \text{r}(A),\text{r}(B)}\}$

# 3. 两洞分块的秩

>[!prp] 命题：$\text{r}\begin{pmatrix} A&O\\O&B \end{pmatrix}=\text{r}(A)+\text{r}(B)$

# 4. 一洞分块的秩

>[!prp] 命题：$\text{r}\begin{pmatrix} A&C\\O&B \end{pmatrix}\geqslant \text{r}(A)+\text{r}(B)$，$\text{r}\begin{pmatrix} A&O\\D&B \end{pmatrix}\geqslant \text{r}(A)+\text{r}(B)$

# 5. 行列分块的秩

>[!prp] 命题：$\text{r}(A\mid B)\leqslant \text{r}(A)+\text{r}(B)$，$\text{r}\begin{pmatrix} A\\B \end{pmatrix}\leqslant \text{r}(A)+\text{r}(B)$

# 6. 矩阵加减法的秩

>[!prp] 命题：$\text{r}(A+B)\leqslant \text{r}(A)+\text{r}(B)$，$\text{r}(A-B)\leqslant \text{r}(A)+\text{r}(B)$
>无论矩阵加减，总小于等于秩的加

# 7. 矩阵减法的秩

>[!prp] 命题：$\text{r}(A-B)\geqslant |\text{r}(A)-\text{r}(B)|$
>矩阵减法的秩总是大于等于秩的距离


# 8. Sylvester 不等式

>[!thm] Sylvester 不等式
>设 $A$ 是 $m\times n$ 矩阵，$B$ 是 $n\times t$ 矩阵，成立下列不等式：
>$$\text{r}(A)+\text{r}(B)-n \leqslant \text{r}(AB)\leqslant \min \{{ \text{r}(A),\text{r}(B)}\}$$ 




# 9. 秩的降阶公式

>[!prp] 命题：秩的降阶公式
设有分块矩阵 $M=\begin{pmatrix} A&B\\C&D \end{pmatrix}$，
（1）若 $A$ 可逆，则 $$\text{r}(M)=\text{r}(A)+\text{r}(D-CA^{-1}B),$$
（2）若 $D$ 可逆，则 $$\text{r}(M)=\text{r}(D)+\text{r}(A-BD^{-1}C).$$
